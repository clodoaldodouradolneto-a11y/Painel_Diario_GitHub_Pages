# Publicação no GitHub Pages

Este pacote já contém o fluxo automático em:

`.github/workflows/publicar.yml`

Após enviar os arquivos ao repositório:

1. Abra **Settings**.
2. Entre em **Pages**.
3. Em **Build and deployment > Source**, selecione **GitHub Actions**.
4. Abra a guia **Actions** e aguarde o fluxo **Publicar Painel no GitHub Pages**.
5. O endereço publicado aparecerá no resultado da execução e em **Settings > Pages**.

Importante: envie o conteúdo da pasta para a raiz do repositório, de modo que `index.html` fique na primeira tela do repositório.
